# Demo Taller Atributos
  
Crear un proyecto de nombre `demo-taller-atributos`, crear un paquete de nombre `dto`, en el cual se deben crear las siguientes clases. 

>por cada 2 clases subir los cambios al repositorio *demo-taller-atributos* en github

1. crea una la clase de nombre `Perro` agregar atributos de tipo `[texto, número y decimales]`
2. crea una la clase de nombre `Gato` agregar atributos de tipo `[texto, número y decimales]`
3. crea una la clase de nombre `Leon` agregar atributos de tipo `[texto, número y decimales]`
4. crea una la clase de nombre `Coche` agregar atributos de tipo `[texto, número y decimales]`
5. crea una la clase de nombre `Barco` agregar atributos de tipo `[texto, número y decimales]`
6. crea una la clase de nombre `Bicicleta` agregar atributos de tipo `[texto, número y decimales]`
7. crea una la clase de nombre `Persona` agregar atributos de tipo `[texto, número y decimales]`
8. crea una la clase de nombre `Producto` agregar atributos de tipo `[texto, número y decimales]`
